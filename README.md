# HooBank - Modern UI/UX website using React.js & Tailwind CSS

[DEMO](https://vasyl-pavlenko.github.io/bank/)

 Welcome to the repository for our Landing Page project built using Tailwind, HTML, CSS, and React. This project showcases a modern and responsive landing page design implemented with best practices in mind. Whether you're looking to learn Tailwind, enhance your HTML/CSS skills, or explore React integration, this project is a great starting point.

## Features
- Tailwind
- Responsive Design: The landing page is designed to be responsive and optimized for various screen sizes and devices.
- React Components: Explore the usage of React components to create reusable UI elements.
- Interactive Elements: Implement interactive features using React to enhance user experience.

## Getting Started
Clone this repository to your local machine using:
```bash
git clone https://github.com/vasyl-pavlenko/bank.git
cd bank
npm install
npm run dev
```

### Open your browser and visit http://localhost:5173 to see the website in action!

## Project Structure
- src/components: Contains React components for various sections of the landing page.
- public: Holds static assets such as images and the HTML entry point.
